# 울산 학교 찾기 — 추가 기능 아이디어

에이전트 팀 브레인스토밍 결과 (12개).

## 🏷️ 학교 카드·마커 배지 (기존 데이터 즉시 활용)

1. **공립/사립 아이콘** — 카드 좌측에 공/사 작은 배지 (`founding_type`)
2. **남/여/공학 구분** — 중·고 카드에 ♂/♀/∞ 아이콘 (`coedu`)
3. **역사학교 배지** — 개교 50년 이상 "⭐ 50년+" 골드 배지 (`founded`)
4. **과밀/소규모 경고** — 학급당 30명↑ 🔴, 10명↓ ⚪ 경고 (`avg_students_per_class`)
5. **특수학급 有** — 초록 배지 "특수학급" (`special_classes > 0`)
6. **고등학교 유형 필터** — 일반/특성화/자율 드롭다운 (`hs_category`)

## 🗺️ 지도·필터 편의

7. **주소 입력 → 배정학교 자동 표시** ⭐⭐ — 검색창 옆 "주소로 찾기" 버튼. 학부모 최대 수요. 카카오 지오코딩 API 필요
8. **지도 색상 범례 고정** — 우하단 작은 박스, 줌아웃 시 마커색 헷갈림 방지
9. **최근 본 학교 3개** — 사이드바 상단 얇은 칩 띠 (세션 기반)

## 🏗️ 개발사업 연계 강화

10. **학구 내 개발사업 카운트 배지** — 학교 선택 시 "신규 주택 3건 계획" 배지
11. **착공 상태 색상 구분** — 개발사업 마커: 착공🟢 / 공사중🟠 / 미착공⚫
12. **개발사업 → 배정학교 하이라이트** — 개발사업 클릭 시 `school_elem/mid/high`에 해당하는 학교 목록이 자동 스크롤·강조

## 📊 번외 (구현 중규모)

- 구/군 통계 미니 대시보드 (평균 학생수·공사립 비율 등)
- 필터 조합 북마크 ("우리 동네 초등" 저장)
- 학교 2개 선택 시 지도에 거리 선 그리기
