# 개발사업 표시 기능 설계

## 개요
기존 울산 학교 찾기 지도(index.html)에 개발사업 128건을 오버레이로 표시하는 토글 기능 추가.

## 1. 토글 버튼
- 헤더에 대시보드 버튼 옆 배치
- OFF(기본): 테두리만 있는 일반 버튼 스타일
- ON: 주황색 배경(#F59E0B) 활성화 상태
- ON/OFF 시 개발사업 마커만 추가/제거, 학교 마커에 영향 없음

## 2. 마커 디자인
- 둥근 사각형(border-radius: 6px), 주황 계열(#F59E0B) — 학교 pill과 형태 차별화
- 사업명 약칭 + 세대수 2줄 표시 (줌 14 이상)
- 줌 레벨 낮을 때 세대수만 간략 표시
- 별도 MarkerClusterGroup으로 클러스터링 (학교 클러스터와 독립)

## 3. 팝업/바텀시트
- PC: Leaflet 팝업
- 모바일(768px 이하): 기존 바텀시트 패턴 활용
- 표시 항목: 사업명, 위치, 행정구역, 세대수, 사업기간, 착공여부, 초등학교 배정

## 4. 동작 규칙
- 학교 마커와 개발사업 마커는 독립적으로 동작
- 개발사업 마커 클릭 → 팝업(PC)/바텀시트(모바일) 표시
- 학교 필터(유형) 변경 시 → 개발사업 마커 유지 (토글 상태에 따름)
- 구/군 필터는 개발사업에도 적용 (해당 구/군만 표시)
- 토글 OFF 시 모든 개발사업 마커 제거

## 5. 데이터
- 소스: `data/ulsan_dev_projects.json` (128건)
- 필드: name, district, location, units, period, construction, school_elem, school_mid, school_high, lat, lng

## 6. 구현 범위
- index.html 단일 파일 내 수정 (CSS + JS 인라인)
- deploy/index.html 동기화
